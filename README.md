#  Basic Network Scanning with Nmap

##  Introduction:

**Nmap (Network Mapper)** is an open-source tool used for network discovery and security auditing. It helps identify active hosts, open ports, running services, and operating systems on a network.

---

## commands:

nmap 10.192.242.97

nmap -sV 10.192.242.97

nmap -p- 10.192.242.97

# Sample Output:

Nmap provides details such as:

Open ports

Service names

Device type

Operating system (if detectable)

##  Conclusion:

Nmap is a powerful and essential tool for anyone interested in networking and cybersecurity. Learning basic scanning techniques is the first step toward mastering network security.
